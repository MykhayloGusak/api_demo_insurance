# DEMO REST API

This project is an demo project build with a progressive Node.js framework like express.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
$ npm run test
```

## Swagger endpoint documentation

```bash
$ http://localhost:3030/docs
```
